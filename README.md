# Number Guessing Game (Python)

## About

This is a simple Python game where the user tries to guess a randomly generated number. The program provides hints like "Too high" or "Too low" to guide the player.

## Features

* Random number generation
* Hints (Too high / Too low)
* Tracks number of attempts
* Multiple difficulty levels
* Replay option

## Tech Used

* Python

## How to Run

1. Run Game.py
2. Select difficulty level
3. Start guessing the number
